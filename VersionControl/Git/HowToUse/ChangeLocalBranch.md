# ローカルのブランチを切り替える

```bash
git checkout <branch>
```



## About

既にローカルにあるブランチを切り替えるには
```bash
git checkout <branch>
```

リモートからチェックアウトしつつブランチを切り替えるには
```bash
git checkout‘-b <branch>
```
