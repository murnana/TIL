## About
分散型バージョン管理システムの一つ。  
「分散型」と呼ばれる理由は、サーバー上にある履歴をローカルに落とすことで、オフラインであっても、あるいはサーバー上にある履歴がすべて壊れたとしても、ローカル上で作業することができるから。

またオープンソースであり、無料で商用利用が可能なのも特徴。
