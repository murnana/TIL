# メニューにあるコマンドを実行するには

`EditorApplication.ExecuteMenuItem(string)` を呼ぶ。

## Examples

### プロジェクトウィンドウを開く

```cs
EditorApplication.ExecuteMenuItem("Window/General/Project");
```
